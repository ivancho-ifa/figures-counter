int figures_counter::count_figures ( size_t  rows,
size_t  cols,
std::istream &  data 
)

Returns the number of figures in data.

The input is a table with the contents of data, formatted in size rows by cols. The table's contents data consists of only two characters - EMPTY_CELL and FULL_CELL. A figure is consisting of one or more cells with value FULL_CELL that share a side. In example 1 there is one figure and in example 2 there are 2 figures.

colsColumns in the table. A positive integer.
dataContents of the table, formatted in rows rows with length cols. A readable data stream holding rows new line separated rows with cols length each.
Returns
The number of figures in the input.
